翻译文件生成与编译

  • 添加完翻译的标记后，执行以下命令即可在locale文件夹下生成翻译文件：

python manage.py makemessages -l en
python manage.py makemessages -d djangojs -l en

  • 填充完所有翻译后的字符串，就可以执行编译命令生成.mo文件
