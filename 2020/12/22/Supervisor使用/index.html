supervisor是一个python开发的、通用的进程监控、管理工具，可以将一个普通的命令行进程变为后台daemon,并且监控进程状态，异常自动重启。相较于nohup的方式，这种部署方式使用起来更方便简洁，并且运行也更稳定。

1.安装

sudo apt-get install -y supervisor

2.创建项目

cd /etc/supervisor/conf.d/
sudo vim mipin.conf
